📊 OpenAI vs 火山引擎 ARK 格式对比
🟢 OpenAI 格式
{ "model": "gpt-4o-mini", "messages": [ {"role": "user", "content": "你好"} ], "temperature": 0.7 }
🔴 火山引擎 ARK 格式
{ "model": "glm-4-7-251222", "input": [ { "role": "user", "content": [ { "type": "input_text", "text": "你好" } ] } ] }
⚠️ 关键区别：
  • OpenAI: messages + content: "字符串"
  • ARK: input + content: [{type: "input_text", text: "..."}]
